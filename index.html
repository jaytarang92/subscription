Need a subscription? But too lazy to type?.

This program allows for automated creation of online accounts. It provides the template just needs to know where to put it. This was a fun weekend project that helped me understand a lot of programming basics. Hopefully you guys will find this usefull.

API's Used

Twilio for sending text messages https://www.twilio.com/api

Selenium for sending keys to browser http://www.seleniumhq.org

lxml/html for parsing the html pages http://lxml.de
